## Overview

java_Tennis_Project is a comprehensive Java-based simulation platform designed to model and manage grand slam tennis tournaments with realism and flexibility. It provides a structured architecture for orchestrating player registration, match progression, and statistical tracking, making it ideal for developers building sports management applications.

## Getting Started

### Prerequisites

This project requires the following dependencies:

*   **Programming Language**: Java
*   **Package Manager**: Maven

### Installation

Build java_Tennis_Project from the source and install dependencies:

1.  Clone the repository:

    ```bash
    git clone https://github.com/Salah-eddine-boudi/java_Tennis_Project
    ```

2.  Navigate to the project directory:

    ```bash
    cd java_Tennis_Project
    ```

3.  Install the dependencies:

    Using [maven](https://maven.apache.org/):

    ```bash
    mvn install
    ```
